# Sistema de Gestión de Pedidos y Stock
Proyecto desarrollado en Java orientado a objetos.

El sistema permite:
- gestionar productos
- controlar stock
- registrar clientes
- generar pedidos

El objetivo del proyecto es practicar lógica de programación, POO y modelado de sistemas reales.

## Funcionalidades

- Ver información de productos
- Agregar stock
- Crear pedidos
- Validar stock disponible
- Calcular total de compra
- Relacionar clientes con pedidos

- ## Tecnologías utilizadas

- Java
- Programación Orientada a Objetos (POO)

- ## Clases principales

- Cliente
- Producto
- Pedido
- Main

- ## Objetivos de aprendizaje

- Practicar Programación Orientada a Objetos
- Entender relaciones entre objetos
- Modelar lógica de negocio
- Construir sistemas escalables
